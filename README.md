## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/ritalisboar/desafio-hackathon-decola.git
```

Entre no diretório do projeto

```bash
  cd desafio-hackathon-decola
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
